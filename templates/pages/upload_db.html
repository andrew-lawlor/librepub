Upload Kobo Database

Use the file browser to upload your Kobo's 'KoboReader.sqlite' file.

Need help finding it?

  1. Plug your Kobo device into your computer.
  2. Open your Kobo using your file manager (e.g., Windows Explorer).
  3. Toggle the 'Show Hidden Files' option in your operating system. Here's how to do that in Windows and Mac. On Linux, press CTRL + H.
  4. Click 'Browse' below. Find your Kobo eReader. Click into the '.kobo' folder, in which you'll find the 'KoboReader.sqlite' file. Select this file and confirm!

But What About My Privacy?

Excellent question! LibrePub simply extracts your notes, highlights and saved vocabulary. Your database file is stored temporarily, processed, and then immediately deleted. You can export your data at any time in universal, plain-text formats. No lock-in here!

Don't believe us? Read the source code for yourself and see! That's the beauty of free software.

You can always pull the project and run it on localhost. It's only a single binary ;)
